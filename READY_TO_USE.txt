📊 STATISTICS DASHBOARD
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Color-coded stats appear after verification:

🟢 VERIFIED (Green)
   Claim is factually accurate
   Example: "Water boils at 100°C"

🟡 INACCURATE (Yellow)
   Claim has some errors
   Example: "Water boils at 95°C"

🔴 FALSE (Red)
   Claim is wrong
   Example: "Water boils at 50°C"

⚫ UNVERIFIABLE (Gray)
   Cannot determine
   Example: "UFOs exist"

"Upload not working?"
→ Make sure it's a PDF file
→ Try a smaller file first
→ Check internet connection

"Dark mode not showing?"
→ Refresh page (Ctrl+Shift+R)
→ Clear browser cache
→ Check if Moon/Sun button visible in top-right

"Results taking too long?"
→ Normal for complex documents
→ Check internet connection
→ Try a shorter document

"API Error?"
→ Verify API key in .env.local
→ Check if OpenAI account is active
→ Verify account has API credits
