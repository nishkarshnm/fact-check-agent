STEP 1: Open the App
────────────────────
  1. Go to: http://localhost:3000
  2. You should see FactCheck landing page
  3. Scroll down to find upload area

STEP 2: Test Upload (DEMO MODE)
─────────────────────────────────
  1. Drag a PDF file onto the upload area OR
     Click "Upload" button and select a PDF
  2. The app extracts text and shows DEMO results
  3. Demo shows 3 sample claims with verification status
  4. You'll see: VERIFIED, INACCURATE, FALSE results
  5. Each result has evidence and confidence score

STEP 3: Toggle Dark Mode
────────────────────────
  1. Look at TOP-RIGHT of page
  2. Find the icon:
     • Moon icon = Light mode (click to go dark)
     • Sun icon = Dark mode (click to go light)
  3. Click it - theme changes instantly!
  4. Refresh page - your choice is remembered
